# node-tools
node 实用工具库存 🔧


## list

### read.js
* 执行： `node read.js`
* 功能： 
    * 读取 yaml 文件
    * 读取 excel 文件

### save.js
* 执行： `node save.js`
* 功能： 将数据保存为 yaml + json 格式文件

### excelToJson.js
* 执行： `node excelToJson.js`
* 功能： 将 excel 转换为 json 格式文件并保存
